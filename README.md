# Seoul Restaurant - Site Web

Site web moderne pour **Seoul Restaurant Coréen**, situé à Douala, Cameroun.

## 🍽️ Fonctionnalités

- **Menu complet** avec toutes les catégories (Barbecue, Sushi, Entrées, Grillades, Fritures, Riz, Pâtes, Suppléments)
- **Multilingue** : Français, 한국어, English, 中文
- **Réservation en ligne** via WhatsApp
- **Design premium** inspiré de la gastronomie haut de gamme
- **Responsive** : optimisé mobile, tablette et desktop

- React + TypeScript

## 🚀 Installation

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```

Le site compilé se trouve dans le dossier `dist/`.
